# 高德扫街榜侧边导航栏设计规范

## 研究成果总结

基于对高德扫街榜移动端界面的采集和分析，以下是侧边导航栏的核心设计规范：

### 1. 一级标题（Primary Categories）

**设计特征：**
- **字体**：PingFang SC / 黑体（Bold Sans-serif）
- **字号**：14px - 16px（相对较小且精致）
- **字重**：500-600（中等偏粗）
- **颜色**：深灰色 #333333 或 #2C2C2C（非纯黑，保持温度感）
- **行高**：1.4 - 1.5
- **装饰**：**无任何装饰符号**（高德已移除了所有装饰）
- **间距**：一级标题之间的垂直间距约 12px - 16px
- **交互状态**：
  - 默认：深灰色文本
  - 选中/悬停：品牌色（#FF5500 或类似的活力橙）
  - 背景：可选择浅色背景或仅文本颜色变化

### 2. 二级标题（Secondary Items）

**设计特征：**
- **字体**：PingFang SC / 微软雅黑（Regular Sans-serif）
- **字号**：12px - 13px（明显小于一级标题）
- **字重**：400（常规）
- **颜色**：中灰色 #666666 或 #757575（相比一级标题更浅）
- **行高**：1.3 - 1.4
- **展开/收起**：支持点击展开和收起（不像高德一直展开）
- **间距**：
  - 二级标题与一级标题的距离：6px - 8px
  - 二级标题之间的距离：4px - 6px
  - 二级标题的左缩进：8px - 12px
- **交互状态**：
  - 默认：中灰色文本
  - 选中：品牌色 #FF5500
  - 首个二级项可选择使用胶囊样式（如"两人约会"）

### 3. 胶囊样式（Capsule Style）

**应用场景：**
- 仅应用于首个二级菜单项（如"两人约会"）
- 其余二级菜单项保持纯文本

**设计规范：**
- **背景色**：品牌色 #FF5500 或浅品牌色 #FFE6D5
- **文本色**：白色 #FFFFFF（如果背景是 #FF5500）或品牌色（如果背景是浅色）
- **内边距**：4px 8px
- **圆角**：12px - 16px（完全圆角）
- **字号**：12px（与其他二级标题一致）
- **字重**：500（相比其他二级标题稍粗）

### 4. 动效与交互

**展开/收起动画：**
- 动画时长：300ms - 400ms
- 缓动函数：ease-in-out 或 cubic-bezier(0.4, 0, 0.2, 1)
- 效果：平滑的高度变化，可配合 opacity 渐变

**悬停效果：**
- 背景色轻微变化（如果有背景）
- 文本颜色变化为品牌色
- 无明显阴影或其他重型效果

### 5. 整体布局

**侧边栏宽度：**
- 桌面端：88px - 100px（保持精致感）
- 移动端：80px - 88px（最大化主内容区域）

**背景色：**
- 白色 #FFFFFF 或极浅灰 #F8F8F8
- 与主内容区域保持视觉分离，但不过度

**分割线：**
- 可选择在一级标题之间添加极细分割线（1px，颜色 #E8E8E8）
- 或通过间距和视觉权重实现分离

### 6. 字体与排版系统

**推荐字体堆栈：**
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;
```

**字号层级：**
- 一级标题：14px - 16px（font-weight: 500-600）
- 二级标题：12px - 13px（font-weight: 400）
- 标签/标记：11px - 12px（font-weight: 400）

### 7. 颜色系统

**核心色：**
- 品牌色（选中/活跃）：#FF5500
- 文本主色（一级）：#333333
- 文本次色（二级）：#666666
- 文本弱色（禁用）：#999999
- 背景色：#FFFFFF 或 #F8F8F8
- 分割线：#E8E8E8

### 8. 对比高德的改进

**FIND ME 相比高德的差异化设计：**
1. **可折叠二级菜单**：高德一直展开，FIND ME 支持点击展开/收起
2. **团购套餐内容**：针对社交消费场景优化（闺蜜套餐、兄弟套餐、情趣套餐等）
3. **动效交互**：保留现有的精致动效设计
4. **响应式设计**：在移动端采用抽屉菜单，桌面端保持侧边栏

## 实现建议

1. **字体加载**：使用系统字体堆栈，避免额外的字体加载
2. **性能优化**：使用 CSS 变量管理颜色和间距，便于主题切换
3. **可访问性**：确保充足的颜色对比度，支持键盘导航
4. **响应式**：在不同屏幕尺寸下测试，确保文本不被截断
